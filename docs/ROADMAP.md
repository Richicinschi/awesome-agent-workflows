# Workflow Roadmap

This roadmap keeps the project contribution-friendly without encouraging low-quality prompt dumping.

## High-priority additions

- Real transcript examples for the top 10 workflows.
- Demo GIF for the README.
- More tool-specific adapter notes for Gemini CLI and framework agents.
- Copy-paste task briefs for common stacks: Python, Node, web UI, docs-only repos.
- A small static site or generated index from `agent-workflows.json`.

## Contribution standard

A new card should include:

- clear trigger;
- copy-paste brief;
- bounded steps;
- explicit verification;
- failure modes;
- human gate;
- no secrets or private data.

## Quantity target

The project should grow by useful clusters, not random volume:

- 40 cards: broad foundation.
- 60 cards: common engineering operations covered.
- 100 cards: mature catalog with examples and tool adapters.
