# Local/self-hosted agents

Use the cards as local operating procedures for private agents running with local models or tool servers.

## Best starting workflows

- Useful for privacy-sensitive workflows and local AI stacks.
- Combine Local AI Stack Setup with MCP and Tool Integration Review.
- Keep model limitations visible in the handoff.

## Gotchas

- Local models may skip details or miss subtle security issues.
- Tool permissions matter more than model location.
