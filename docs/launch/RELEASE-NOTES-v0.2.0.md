# v0.2.0 — Expanded workflow library

This release turns the initial workflow seed into a broader practical catalog.

Highlights:

- Adds 30 new workflow cards, bringing the library to 40 workflows.
- Adds a richer scenario-based workflow catalog.
- Expands `agent-workflows.json` with trigger, risk level, tools, tags, and verification metadata.
- Adds tool adapters for Claude Code, Codex CLI, Cursor, Aider, OpenCode, and local/self-hosted agents.
- Adds an agent tool comparison matrix.
- Adds a worked example showing fresh-context review catching an unverified PR.
- Adds a workflow roadmap and stronger curation policy examples.
- Adds example-led launch copy for ethical growth.
