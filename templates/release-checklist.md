# Release Checklist

## Readiness

- [ ] Tests/build/lint pass from a fresh checkout.
- [ ] Security-sensitive changes reviewed.
- [ ] README/docs match current behavior.
- [ ] Install or quick-start path works.
- [ ] License and contribution path are present.

## Rollout

- [ ] Release notes are written.
- [ ] Demo/screenshot/sample output is ready.
- [ ] First users know what problem this solves.
- [ ] Rollback or unpublish path is known.

## First-hour checks

- [ ] Repo page renders correctly.
- [ ] Badges resolve.
- [ ] Issues/PR templates are visible.
- [ ] CI run is green or any failure is explained.
- [ ] Launch post links point to the correct URL.
